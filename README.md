# frontend_kiosk

Frontend for the kiosk application.

## Features

- Built with TypeScript
- Handles the kiosk user interface
- Bundled using the `dist/` directory

## Getting Started

1 **Install dependencies**
```bash
npm install
```
2 **Run APP**
```bash
cd .\kisok\
npm run dev
```

*Local webpage GUI*
```
http://localhost:3000
```

**Project structure**
```
.
├── kiosk/             # Source code
├── dist/              # Build output
├── package.json
├── package-lock.json
```
